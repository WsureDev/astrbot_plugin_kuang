# astrbot_plugin_kuang

AstrBot 插件“框”。

触发方式：

- 发送单字 `框` 并附带图片
- 回复带图消息发送单字 `框`
- 单独发送单字 `框`，此时自动使用发送者头像

效果说明：

- 每张图固定绘制 5 个白色线框
- 优先使用识别框，当前首版内置了整人/上半身/头部与猫脸的 OpenCV 识别
- 识别不足 5 个时自动补随机框
- 随机框按 FPS 外挂透视框风格生成，默认使用偏细长的矩形比例

当前实现策略：

- 人体优先级：整个人 > 上半身 > 头部
- 动物/角色暂以 OpenCV 自带猫脸和类人脸结果兜底，后续可以替换成 YOLO 等更强检测器
- 多张图会逐张处理并分别回图

依赖安装：

```bash
pip install -r requirements.txt
```
